# To Do List

A simple, responsive to-do list application for managing daily tasks.

**[Live Demo](https://tbarnett42.github.io/To-Do-List/)**

## Features

- Add, complete, and delete tasks
- Local storage persistence
- Clean, intuitive interface

## Technologies

HTML5 • CSS3 • JavaScript

## Installation
```bash
git clone https://github.com/tbarnett42/To-Do-List.git
cd To-Do-List
```

Open `index.html` in your browser.
